# Data Dictionary

## `data/train_test/`

Each position has a training set (historical draft classes) and a 14-player
test set (2026 prospects). Training data covers draft classes **2010-2024**.

| File | Rows | Description |
|---|---|---|
| `QB_train.csv` | 164 | Historical QB prospects, 2010-2024 draft classes |
| `QB_Test.csv` | 14 | 2026 QB prospects to project |
| `RB_train.csv` | 278 | Historical RB prospects, 2010-2024 draft classes |
| `RB_Test.csv` | 14 | 2026 RB prospects to project |
| `WR_train.csv` | 436 | Historical WR prospects, 2010-2024 draft classes |
| `WR_Test.csv` | 14 | 2026 WR prospects to project |

### Common columns (all positions)

| Column | Description |
|---|---|
| `college_team` | School the prospect played for in their final college season |
| `college_conference` | Conference of `college_team` |
| `nfl_team` | NFL team that drafted the player (training data only) |
| `year` | NFL draft year |
| `overall` | Overall pick number (model target for `pred_pick_*`) |
| `round` | Draft round |
| `pick` | Pick number within the round |
| `name` | Player name |
| `position` | Player position |
| `height` | Height (inches) |
| `weight` | Weight (lbs) |
| `season_before_draft` | College season the stats below are drawn from (the year before the draft) |
| `regular_season_games` | College team's regular season games played |
| `regular_season_wins` / `regular_season_losses` | College team's regular season record |
| `postseason_games`, `postseason_wins`, `postseason_losses` | College team's postseason results |
| `forty`, `bench`, `vertical`, `broad_jump`, `cone`, `shuttle` | NFL Combine measurables (often missing for players who didn't test) |
| `attended_combine` | Whether the player attended the NFL Combine |

### QB-specific columns

| Column | Description |
|---|---|
| `passing_completions`, `passing_att`, `completion_pct` | Season completions, attempts, completion % |
| `passing_yds`, `passing_td`, `passing_int` | Season passing yards, TDs, interceptions |
| `passing_ypa` | Passing yards per attempt |
| `rushing_car`, `rushing_yds`, `rushing_td`, `rushing_ypc` | Season rushing carries, yards, TDs, yards per carry |

### RB-specific columns

| Column | Description |
|---|---|
| `rushing_car`, `rushing_yds`, `rushing_td`, `rushing_ypc` | Season rushing carries, yards, TDs, yards per carry |
| `receiving_rec`, `receiving_yds`, `receiving_td`, `receiving_ypr` | Season receptions, receiving yards, TDs, yards per reception |

### WR-specific columns

| Column | Description |
|---|---|
| `receiving_rec`, `receiving_yds`, `receiving_td`, `receiving_ypr` | Season receptions, receiving yards, TDs, yards per reception |

## `data/team_strength/`

`R2009.csv` through `R2025.csv` — one file per college football season, exported
from Sports-Reference team ratings. Each covers the season ending in that
year (e.g. `R2025.csv` is the 2025 college football season, used as
`season_before_draft = 2025` for 2026 draft prospects).

**These files have two header rows** — a category row, then the real column
names — so they must be read with `skip = 1`. Key columns used by the models:

| Column | Description |
|---|---|
| `School` | College team name (joined against `college_team`) |
| `SRS` | Simple Rating System — overall team strength relative to schedule |
| `OSRS` | Offensive SRS — offensive component of SRS |

The Sports-Reference layout repeats the `SRS` header across the
Overall/Offense/Defense sections (`SRS`, `SRS.1`, `SRS.2`), so
`R/utils/team_strength.R` picks the first matching column. Teams that don't
appear in a given year's file (e.g. small-school transfers, FCS schools) are
assigned the 10th-percentile SRS/OSRS for that year rather than `NA`.

## `output/results/`

Generated by the model scripts (`R/qb/qb_model.R`, `R/rb/rb_model.R`,
`R/wr/wr_model.R`). Each `*_Final_Draft_Results.csv` is the 2026 test set
plus the merged team-strength columns, per-game stats, and:

| Column | Description |
|---|---|
| `pred_pick_lm` | Projected overall pick from the linear regression model |
| `pred_pick_rf` | Projected overall pick from the random forest model |
| `prob_1st_round` | Probability of being picked in round 1, from logistic regression |

`RB_2026_Draft_Predictions.csv` and `WR_2026_Draft_Predictions.csv` are
outputs of the earlier archived models (`R/archive/rb_model_primary.R`,
`R/archive/wr_model_primary.R`), kept for reference.
